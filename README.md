# Deepfake Detection System

A deepfake image detection system built using Xception, PyTorch, Flask, and React. The system analyzes uploaded images and classifies them as **Real** or **Fake**, along with a confidence score.

## Features

* Deepfake image detection using Xception
* Real/Fake image classification
* Confidence score prediction
* Flask REST API backend
* React frontend with image upload functionality
* Real-time inference

## Tech Stack

### Machine Learning

## Dataset

The model was trained on a balanced dataset created from multiple sources:

### Real Images

* FaceForensics++ (Real)
* CelebA
* StyleGAN2 (Real)

### Fake Images

* FaceForensics++ (Fake)
* StyleGAN2 (Fake)
* Diffusion Generated Faces

### Dataset Statistics

| Split      | Real  | Fake  |
| ---------- | ----- | ----- |
| Train      | 9,000 | 9,000 |
| Validation | 1,500 | 1,500 |
| Test       | 1,500 | 1,500 |

Total Images: **24,000**

## Model Details

* Architecture: Xception
* Framework: timm
* Loss Function: BCEWithLogitsLoss
* Optimizer: AdamW
* Learning Rate: 0.000005
* Weight Decay: 0.0001

## Results

### Mixed Test Set

* Accuracy: 93.57%
* Precision: 93.31%
* Recall: 93.87%
* F1-Score: 93.59%
* ROC-AUC: 98.49%
